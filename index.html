Types of Malware

  • Remote Access Trojans
  • Ransomwares
  • Viruses
  • Trojans

  • Spyware

Remote Access Trojans

Is a program which a hackers installs on a Victims computer however it grants the Hacker to connect to the Victims Computer giving full access to files, webcams and any other thing, It can be prevented via Port Blocking/Firewall and Checking integrity of source file your installing

Ransomwares

A program which encrypts all files and asks the user to pay for a key to unencrypt their files, Ransom attacks are the biggest attacks to happen and dangerous as it can result in big Data breachs.

Viruses

A Computer program that spreads from computer to computer and it replicates its self and infects programs and apps by injecting its code into applications. They can spread quickly throughout a network.

Trojans

Often appear to be something which is wanted or needed and its purpose, Trojans make copies of its self to steal information or harm their system. To prevent this always check the SHA-256 (hash) Bit and make sure its from an official source.

Worms

Worms use a network to spread. Unlike a virus it doesn’t attach itself to a program. Worms almost always cause harm to a network, even if its just consuming Bandwidth.

Spyware

Spyware in a hidden program which spys on a user which can be a keylogger and gives information back to a third party/hacker. Can be difficult to detect.

Adware is also known as Advertising- supported software. This is any software package which automatically shows adverts, such as pop ups. It may also be in the user interface. It can also include spy ware like keyloggers.

Key Loggers Collect information and send data to a third party/hacker it logs all keys pressed such as Passwords, Credit/Debit card numbers. It comes in 2 forms Malware/Hardware.
